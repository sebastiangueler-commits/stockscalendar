## 🚀 Quick Start

### Prerequisites
- Node.js 16+ 
- Python 3.8+
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone <repository-url>
cd stockscalendar
```

2. **Install Frontend Dependencies**
```bash
cd frontend
npm install
```

3. **Install Backend Dependencies**
```bash
pip install -r requirements.txt
```

### Running the Application

1. **Start the Backend**
```bash
python app.py
```
Backend will run on http://localhost:8001

2. **Start the Frontend**
```bash
cd frontend
npm start
```
Frontend will run on http://localhost:3000

### Production Build

1. **Build Frontend for Production**
```bash
cd frontend
npm run build
```

2. **Serve Production Build**
```bash
npx serve -s build
```

## 🔧 API Endpoints

### Authentication
- `POST /api/auth/login` - User login

### Calendar Data
- `GET /api/calendar/monthly` - Monthly calendar
- `GET /api/calendar/historical` - Historical data
- `GET /api/calendar/fundamental` - Fundamental analysis

### System
- `GET /api/status` - System status
- `POST /api/force-update` - Force analysis update
- `POST /api/daily-update` - Daily update
